Установка:
- python3 -m venv venv
- source venv/bin/activate
- pip3 install -r requirements.txt

Справка:
- python3 main.py help

Запуск:
- python3 main.py

*примечание: перед началом запуска нужно:*
1) Добавить вашего бота в группу
2) Установить токен бота в .env
3) В файле "chat_ids.txt" прописать ТГ чаты(каждый с новой строки)
4) В файле "message.txt" написать необходимый текст
5) Запустить "python3 main.py" для отравки заготовленного текста в указанные чаты
